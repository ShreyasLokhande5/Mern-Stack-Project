A full-stack Online Clothing Shopping Web Application built using the MERN stack, designed to deliver a modern, scalable, and real-world e-commerce experience. This project simulates an industry-level online fashion store with complete user and admin workflows, secure backend APIs, and cloud-based media handling.
The application supports product browsing, user authentication, cart management, and a powerful Admin Dashboard for managing products, users, and orders. It is built with a clean architecture and follows best practices used in real production systems.
